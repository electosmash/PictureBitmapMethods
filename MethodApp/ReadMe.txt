		METHODS AND FUNCTIONS
----------------------------------------------------------
	~Standard Create:"an expanding radius that changes
intensity with each increase in radius..."
-itterations: amount generated
-Sin wave max: Intensity of wave color 
-depth(ex.maxium darkness, maximum brightness)
Theta offset: the speed of waves
@Example"https://imgur.com/gallery/8X3suF3";

	~Over image?:"modifier for rendering images, 
effects are mixed with orginal photo MUST BE CONVERTED 
TO A GRADIENT FIRST"
@Example"https://imgur.com/gallery/Wb5dcux";

	~Clock?:"modifier for rendering images,
rotates around the center creating a interesting pattern,
use only up to 360"
@Example"https://imgur.com/gallery/nWWWmu6";

	~Color?"modifier for rendering images,
adds some funky colors!"
----------------------------------------------------------
	Standard Gradient:"Conversion to grayscale"
-itterations: only works with gradient change...
-sin wave max: itensity of gradient render
-theta offset: nothing
-radius: amount of distance to change
@Example"https://imgur.com/gallery/cSzUakf";

	~Gradient change:"Changes gradient conversion,
into a multi rendering method"
@Example"https://imgur.com/gallery/FT9PC1W"

	~Color:"Changes from black and white to color,
does not need to be converted to gradient for this one.
when used with gradient change creates some pretty colorful
results.."
@Example"https://imgur.com/gallery/KiHzHUK"
